### 实验操作考试题目：（回忆版）
数码管显示低四位开关所表示的十六进制数，要求：
* 按下C按钮时，数码管显示数字；
* 接着按下L按钮，数字以1Hz的频率左移，当左移到数码管最高位时，接下来回到数码管最低位继续左移（循环功能）
* 再次按下L按钮，数字以1Hz的频率右移，当右移到数码管最低位时，接下来回到数码管最高位继续右移。
* 报告要求
    + 写出硬件框图，软件算法，软件流程图，简要说明工作原理。
